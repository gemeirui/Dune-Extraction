This study provides the ArcGIS Toolbox “Dune Extraction Model.atbx” for dune extraction, DEM data of three types of dunes used for testing, add it to the geoprocessing toolbox, and right-click to edit to use it.See the model instruction guide for details.

Model inputs include:
1. DEM data
2. Window size
3. Slope cost distance threshold

Model output includes:
1. Slope cost distance threshold raster
2. Sand dune extraction result vector file
